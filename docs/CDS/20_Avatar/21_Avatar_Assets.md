## 三、素材制作规范

### 3.1 SVG 规范

| 属性 | 规范 |
|------|------|
| viewBox | 0 0 200 200 |
| 描边色 | #5C3D2E |
| 描边宽度 | 2.5px |
| 填充色 | 通过 props 传入 |
| 无投影 | 不使用 filter/shadow |
| 无渐变 | 扁平风格，纯色填充 |

### 3.2 颜色插槽

每个 SVG 组件必须支持以下颜色插槽：

| 插槽 | 说明 | 默认值 |
|------|------|--------|
| skinColor | 肤色 | #FFD5B8 |
| hairColor | 发色 | #C68B59 |
| clothingColor | 服色 | #D44A4A |
| outlineColor | 描边 | #5C3D2E |

### 3.3 组件接口

```typescript
interface AvatarPartProps {
  color?: string;        // 主要颜色
  outlineColor?: string; // 描边颜色
  size?: number;         // 尺寸（默认200）
}

// 每个素材都是一个 React 组件
const FaceRound: React.FC<AvatarPartProps> = ({ color, outlineColor }) => (
  <circle cx={100} cy={48} r={28} 
    fill={color} 
    stroke={outlineColor || '#5C3D2E'} 
    strokeWidth={2.5} />
);
```
