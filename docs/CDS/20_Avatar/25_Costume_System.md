# 25 — 服装系统 (Costume System)

> **Companion 服装系统：为每个角色穿上合适的衣服**

---

## 一、服装系统概述

### 1.1 设计目标

- 覆盖日常、职业、季节、特殊场合
- 支持颜色自定义
- 未来支持搭配系统
- 与成长系统联动

## 四、颜色自定义

### 4.1 服装颜色规则

| 规则 | 说明 |
|------|------|
| 主色 | 服装主体颜色，可自定义 |
| 辅色 | 装饰/图案颜色，部分可自定义 |
| 描边 | 统一 #5C3D2E，不可修改 |

### 4.2 预设服装色

```
#D44A4A 红色（默认）
#5B8DBE 蓝色
#6B9E6B 绿色
#E8C872 黄色
#9C7EB5 紫色
#E8734A 橙色
#F5F5F5 白色
#424242 黑色
```

## 五、数据模型

```typescript
interface CostumeConfig {
  id: string;
  name: string;
  category: 'daily' | 'work' | 'season' | 'special' | 'cultural';
  gender: 'male' | 'female' | 'unisex';
  season?: 'spring' | 'summer' | 'autumn' | 'winter';
  unlockLevel?: number;   // 解锁等级
  unlockAchievement?: string; // 解锁成就
  primaryColor: string;   // 主色
  secondaryColor?: string; // 辅色
  svg: React.FC<CostumeProps>;
}
```
