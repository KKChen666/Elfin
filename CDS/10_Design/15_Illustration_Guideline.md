# 15 — 插画规范 (Illustration Guideline)

> **Companion 的视觉风格：Q版、温暖、可爱**

---

## 一、插画风格定义

### 1.1 核心风格

Companion 的插画风格是 **Q版（Chibi）扁平风格**：

| 属性 | 规范 |
|------|------|
| 风格 | Q版/Chibi 扁平 |
| 头身比 | 1:1 ~ 1:1.5（大头小身） |
| 描边 | 2.5px，色值 #5C3D2E |
| 配色 | 暖色为主，饱和度中等 |
| 阴影 | 无投影或极淡投影 |
| 细节 | 简化，保留特征 |

### 1.2 设计灵感

```
┌─────────────────────────────────────┐
│                                     │
│  参考风格：                          │
│  - LINE Friends                    │
```

## 二、头像插画规范

### 2.1 SVG 渲染规格

| 属性 | 值 |
|------|-----|
| viewBox | 200×200 |
| 渲染方式 | SVG 组件组合 |
| 描边色 | #5C3D2E |
| 描边宽度 | 2.5px |
| 裁剪方式 | 圆形（配合 rounded-full） |

### 2.2 渲染顺序

```
1. 发后层 (Hair Back)
2. 身体服装 (Body & Clothing)
3. 腿部 (Legs)
4. 鞋子 (Shoes)
5. 头部 (Head/Face)
6. 发前层 (Hair Front)
7. 眼睛 (Eyes)
8. 嘴巴 (Mouth)
9. 腮红 (Blush)
10. 配饰 (Accessory)
```

### 2.3 素材规格

| 部件 | 当前数量 | V2.0 目标 | 说明 |
|------|----------|-----------|------|
| 脸型 | 5种 | 8种 | 圆脸/鹅蛋/圆宽/方圆/心形 |
| 发型 | 10种 | 30种 | 男5+女5 → 更多 |
| 眼睛 | 6种 | 15种 | 基础表情 |
| 嘴巴 | 5种 | 12种 | 基础表情 |
| 服装 | 8种 | 20种 | 日常/职业/特殊 |
| 配饰 | 6种 | 15种 | 帽子/眼镜/耳环等 |

## 三、空状态插画

### 3.1 设计原则

- 使用 Q 版小人作为主角
- 与当前场景相关
- 带有鼓励性文案
- 可以有轻微动画

### 3.2 空状态列表

| 场景 | 插画描述 | 文案 |
|------|----------|------|
| 无亲友 | Q版小人招手 | "这里还空空的呢，要不要添加一个重要的TA？" |
| 无提醒 | Q版小人日历 | "还没有设置提醒哦，重要日子不能忘~" |
| 无聊天记录 | Q版小人对话框 | "导入聊天记录，看看TA的说话风格~" |
| 搜索无结果 | Q版小人放大镜 | "没找到呢，换个关键词试试？" |

## 五、图标规范

### 5.1 图标库

- 使用 **Lucide React** 图标库
- 不自定义图标（除非 Lucide 没有）

### 5.2 图标尺寸

| 场景 | 尺寸 | Tailwind |
|------|------|----------|
| 内联文本 | 16px | `w-4 h-4` |
| 按钮/标签 | 20px | `w-5 h-5` |
| 导航/列表 | 24px | `w-6 h-6` |
| 页面标题 | 32px | `w-8 h-8` |

### 5.3 图标颜色

- 默认继承父元素文本颜色
- 品牌色图标：`text-primary`
- 禁用图标：`text-gray-300`

### 5.4 图标圆角

Lucide 图标默认为 rounded 风格，与 Companion 设计语言一致。

## 七、插画制作规范

### 7.1 SVG 文件规范

```xml
<!-- 标准 SVG 头像模板 -->
<svg viewBox="0 0 200 200" xmlns="http://www.w3.org/2000/svg">
  <!-- 描边色和宽度 -->
  <!-- 描边色: #5C3D2E -->
  <!-- 描边宽度: 2.5px -->
  
  <!-- 渲染层级 -->
  <g class="hair-back">...</g>
  <g class="body">...</g>
  <g class="head">...</g>
  <g class="hair-front">...</g>
  <g class="eyes">...</g>
  <g class="mouth">...</g>
  <g class="blush">...</g>
  <g class="accessory">...</g>
</svg>
```

### 7.2 导出规范

| 格式 | 用途 | 说明 |
|------|------|------|
| SVG | 头像组件 | 矢量，可缩放 |
| PNG | 静态插画 | @1x @2x @3x |
| Lottie JSON | 动画插画 | 30fps, ≤ 50KB |

## 八、一致性检查

| 检查项 | 要求 |
|--------|------|
| 描边 | 统一 2.5px, #5C3D2E |
| 配色 | 使用设计令牌中的颜色 |
| 风格 | Q版扁平，大头小身 |
| 圆角 | 所有形状使用大圆角 |
| 主题 | 支持深色/浅色模式 |
